# Hessians of the link function

`link_Hessian` function for Hessians of the link function

## Usage

``` r
link_Hessian(t, method = c("logistic", "poisson"))
```

## Arguments

- t:

  t

- method:

  indicates the method to be used for M-estimation. Options include
  "mean", "quantile", "ols", "logistic", and "poisson".

## Value

Hessians of the link function

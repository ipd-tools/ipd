# Show an ipd object

Display a concise summary of an `ipd` S4 object, including method,
model, formula, and a glm-style coefficient table.

## Usage

``` r
# S4 method for class 'ipd'
show(object)
```

## Arguments

- object:

  An object of S4 class `ipd`.

## Value

Invisibly returns `object` after printing.

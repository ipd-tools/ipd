# Gradient of the link function

`link_grad` function for gradient of the link function

## Usage

``` r
link_grad(t, method = c("ols", "logistic", "poisson"))
```

## Arguments

- t:

  t

- method:

  indicates the method to be used for M-estimation. Options include
  "mean", "quantile", "ols", "logistic", and "poisson".

## Value

gradient of the link function
